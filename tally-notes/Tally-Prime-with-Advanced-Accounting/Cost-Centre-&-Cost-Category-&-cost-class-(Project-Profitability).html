1–3. Theory: Cost Centre, Cost Category & Cost Centre Class

Cost Centre

Unit for tracking income & expense

  • Projects, Departments, Branches
  • Measures profitability

Cost Category

Grouping of cost centres

  • Allows multi-dimensional allocation
  • Project + Department in same voucher

Cost Centre Class

Auto-allocation tool

  • Splits amount by % automatically
  • Used for Salary, Rent, Electricity

4. Configuration – Must Enable All Three

F11 → Accounting Features →

  • Maintain Cost Centres → YES
  • Use Cost Categories → YES
  • Use Cost Centre Classes → YES
