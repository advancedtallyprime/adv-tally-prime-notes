1. Introduction – What are Delivery & Receipt Notes?

VoucherPurpose
Delivery Note (DN)Records goods delivered to customer
Receipt Note (RN)Records goods received from supplier

These are Inventory Vouchers — update stock but do not affect accounting until Sales/Purchase is recorded.

2. Step-by-Step Flow

Outward Supply: Sales Order → Delivery Note → Sales Voucher

Inward Supply: Purchase Order → Receipt Note → Purchase Voucher

Benefits: Accurate stock tracking, dispatch vs billing separation, warehouse-friendly

2. Enabling DN/RN Vouchers — Practical Setup

Gateway of Tally → Masters → Alter → Accounting Masters → Voucher Type → Delivery Note → Activate this Voucher Type → Yes

Alternate Way

Gateway of Tally → Transactions → Vouchers → F10/Other Vouchers → Show Inactive → Inventory Vouchers → Delivery Note → Activate → Yes


Same for Receipt Note

7. Linking DN/RN to Sales/Purchase Vouchers

When creating Sales Voucher:
→ Choose “Bring All Pending Tracking Numbers” → Auto-fills items from Delivery Note

When creating Purchase Voucher:
→ Choose “Bring All Pending Tracking Numbers” → Auto-fills from Receipt Note
