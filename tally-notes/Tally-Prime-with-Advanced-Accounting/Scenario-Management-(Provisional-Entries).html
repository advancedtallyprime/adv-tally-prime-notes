1. Theory Section – The Power of What-If Analysis

Optional Vouchers

Expected future entries
Do NOT affect books
Marked with “Optional” tag

Reversing Journals

Provisional adjustments
Auto-reverse after given date
Used for provisions, accruals

Scenario

A reporting layer that includes:
• Actual data
• Optional vouchers
• Reversing journals
• Budgets (optional)

Why Top Companies Use Scenarios?

  • Forecast profit before year-end
  • Plan cash flow for next quarter
  • Show banks projected financials
  • Test impact of big orders
  • Estimate tax liability in advance

2. Practical Step-by-Step (Using ABC TRADERS)

2.1 Enable Features

F11 → Accounting Features →

  • Use Optional Vouchers → YES
  • Use Reversing Journals & Scenarios → YES

2.2 Create Scenario

Name: Forecast-2025
Include Actuals → Yes
Include Optional Vouchers → Yes
Include Reversing Journals → Yes

2.3 Create Optional Vouchers

Expected Sales
Customer1 → 20 Laptops @ ₹60,000 = ₹12,00,000
Press Ctrl + L → Make Optional
Expected Salary (Optional Journal)
Salary Expense Dr 5,00,000 → Outstanding Salary Cr
Make Optional

2.4 Create Reversing Journal

Provision for Maintenance
Provision Expense Dr 1,50,000 → Expense Cr
End of Affect: 31-Mar-2025 → Auto reverses after

2.5 View Scenario Reports

Profit & Loss → Alt+C → Add Column → Select Scenario
See side-by-side: Actual vs Forecast!
