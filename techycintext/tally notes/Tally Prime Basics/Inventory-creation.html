2. Units of Measure (UOM)

2.1 Theory

Tally has 2 types of UOM:

  • Simple Unit: Pcs, Nos, Kg, Litre, Meter, Box
  • Compound Unit: 1 Box = 12 Pcs, 1 Dozen = 12 Nos

2.2 Practical: Create Units

Gateway → Create Master → Units (or Go To → Create → Units)

Simple Unit Example: Symbol: PCS, Formal Name: Pieces

Compound Unit Example: 1 Box = 12 Pcs

3. Stock Groups

3.1 Theory

Stock Groups help classify similar items for reporting and common GST application.

Examples: Electronics, Grocery, Furniture, Clothing

3.2 Practical: Create Stock Group

Gateway → Create Master → Stock Group
Enter Name → Under: Primary → Save with Ctrl + A

3.3 Practical Examples

Group NameItems You Can Store
ElectronicsMobile, Laptop, Printer
GroceryOil, Rice, Sugar
FurnitureChair, Table, Sofa
ClothingShirt, Jeans, T-Shirt

4. Stock Items

4.1 Theory

Stock Item = actual product you buy/sell/maintain stock for.

Required fields: Name, Group, Unit, GST Rate, Opening Qty/Value

4.2 Practical: Create Stock Item

Gateway → Create Master → Stock Item → Fill details → Ctrl + A

4.3 Practical Examples

Example 1: Mobile Phone
Name: Samsung A15
Group: Electronics
Unit: PCS
GST: 18%
